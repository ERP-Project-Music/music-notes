# Identificador de Acordes

Herramienta interactiva para identificar acordes a partir de las notas seleccionadas en el mástil. Muestra el nombre del acorde resultante según la combinación elegida.  
No genera progresiones ni analiza funciones armónicas.

### ¿Cómo usarla?

- Haz clic en el **fretboard** para activar notas. Se detectará automáticamente el acorde.
- Para quitar una nota, haz clic sobre ella nuevamente.

### Consideraciones

- Solo permite acordes de hasta **5 notas**.
- Algunas combinaciones muy complejas podrían no ser reconocidas, ya que el sistema sigue en desarrollo.
- No detecta **inversiones** ni acordes con **drop voicings**.
