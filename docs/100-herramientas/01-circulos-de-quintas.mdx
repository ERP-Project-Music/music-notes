# Círculo de Quintas

El **Círculo de Quintas** es una herramienta visual fundamental en la teoría musical. Organiza las tonalidades mayores y menores según la cantidad de alteraciones (sostenidos ♯ o bemoles ♭) que contienen, facilitando la comprensión de las relaciones armónicas.

Es especialmente útil para guitarristas, ya que permite:

- Memorizar las escalas mayores y sus relativas menores.
- Visualizar cuántas y cuáles alteraciones tiene cada tonalidad.
- Comprender relaciones entre acordes y tonalidades cercanas.
- Improvisar y componer con mayor seguridad y coherencia.

## ¿Por qué se llama "Círculo de Quintas"?

### Sentido horario (a la derecha)

Cada paso hacia la derecha avanza una **quinta justa ascendente**:  
C → G → D → A → E → B → F♯ → C♯

### Sentido antihorario (a la izquierda)

Cada paso hacia la izquierda representa una **quinta descendente** (o cuarta ascendente):  
C → F → B♭ → E♭ → A♭ → D♭ → G♭ → C♭

---

## ¿Para qué sirve el Círculo de Quintas?

- Para **entender la construcción de tonalidades**.
- Para **modular** fácilmente entre escalas cercanas.
- Para **analizar progresiones de acordes** dentro de un contexto tonal.
- Para **entrenar el oído** y reconocer relaciones armónicas.
- Para **visualizar la lógica detrás de las alteraciones** en la armadura.

---

## Tip práctico para guitarristas

Ubicar los acordes del círculo de quintas sobre el mástil de la guitarra te permite:

- Saber qué acordes pertenecen a una tonalidad mayor o menor.
- Reconocer de inmediato las escalas a usar en improvisación.
- Encontrar sustituciones armónicas de forma lógica y musical.
- Conectar posiciones y patrones de escalas entre sí.

---

**En resumen:**  
El Círculo de Quintas no es solo una teoría abstracta. Es una guía visual y práctica que ayuda a tocar, componer e improvisar con mayor comprensión y control sobre la música.
