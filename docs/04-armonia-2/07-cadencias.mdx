# Cadencias

Las **cadencias** son fórmulas armónicas que indican cierres o pausas dentro de una progresión. Al usar **cuatriadas** (acordes con séptima), se enriquecen armónicamente y ofrecen una sonoridad más sofisticada que las triadas.

## Tipos de cadencias

| Tipo             | Fórmula        | Ejemplo (C mayor) | Descripción breve                           |
| ---------------- | -------------- | ----------------- | ------------------------------------------- |
| Perfecta         | V7 - Imaj7     | G7 - Cmaj7        | Cierre fuerte y resolutivo                  |
| Plagal           | IVmaj7 - Imaj7 | Fmaj7 - Cmaj7     | Cierre suave, usado también en música coral |
| Rota (deceptiva) | V7 - VI7       | G7 - Am7          | Sorpresiva, evita la resolución esperada    |
| Semicadencia     | Imaj7 - V7     | Cmaj7 - G7        | Final en tensión, deja la frase abierta     |
| Menor auténtica  | V7 - Im7       | E7 - Am7          | Resolución típica en modo menor             |

## Aplicación

El uso de cuatriadas en las cadencias permite:

- Mayor riqueza armónica.
- Transiciones más suaves o tensas según el contexto.
- Expresión más matizada de cierre o suspensión.

Más adelante se pueden incorporar **tensiones** adicionales o **dominantes secundarios** para ampliar aún más las posibilidades cadenciales.
