# Progresiones

Una **progresión armónica** es una secuencia de acordes que genera dirección tonal. Usar **cuatriadas** (acordes con séptima) aporta mayor riqueza y color a las progresiones comunes.

## Tipos comunes de progresiones

| Nombre                   | Fórmula                     | Función general                       |
| ------------------------ | --------------------------- | ------------------------------------- |
| Cadencia II-V-I          | II7 - V7 - Imaj7            | Resolución clásica en jazz            |
| Cadencia rota            | Imaj7 - IVmaj7 - V7 - vi7   | Giro inesperado al vi7                |
| I - vi7 - II7 - V7       | Imaj7 - vi7 - II7 - V7      | Progresión circular                   |
| I - IVmaj7 - V7 - Imaj7  | Imaj7 - IVmaj7 - V7 - Imaj7 | Tensión y reposo equilibrados         |
| Imaj7 - III7 - vi7 - II7 | Enlace cromático            | Movimiento con dominantes secundarios |

## Ejemplos en C mayor

| Progresión                  | Acordes                    |
| --------------------------- | -------------------------- |
| II7 - V7 - Imaj7            | Dm7 - G7 - Cmaj7           |
| Imaj7 - IVmaj7 - V7 - Imaj7 | Cmaj7 - Fmaj7 - G7 - Cmaj7 |
| Cmaj7 - Am7 - Dm7 - G7      | Imaj7 - vi7 - ii7 - V7     |
| Cmaj7 - E7 - Am7 - D7       | Imaj7 - III7 - vi7 - II7   |

## Conclusión

Las progresiones con cuatriadas enriquecen el color armónico y permiten explorar nuevas sonoridades. Son esenciales en géneros como jazz, soul y R&B, y ofrecen una base sólida para la improvisación y la escritura avanzada.
