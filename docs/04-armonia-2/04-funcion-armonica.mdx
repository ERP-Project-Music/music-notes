# Funciones armónicas

En armonía tonal, los acordes de cuatro sonidos (cuatriadas) permiten una clasificación funcional más precisa gracias a la inclusión de la **séptima**. Las funciones armónicas se agrupan en tres zonas:

- **Tónica**: zona de reposo o estabilidad.
- **Subdominante**: zona de preparación o transición.
- **Dominante**: zona de tensión que busca resolución.

## Clasificación funcional en la tonalidad mayor

| Función      | Grados             | Ejemplos en C mayor |
| ------------ | ------------------ | ------------------- |
| Tónica       | Imaj7, VI-7, III-7 | Cmaj7, Am7, Em7     |
| Subdominante | II-7, IVmaj7       | Dm7, Fmaj7          |
| Dominante    | V7, VIIø7          | G7, Bm7♭5           |

## Características generales por función

### Función tónica

| Rasgo                   | Detalle            |
| ----------------------- | ------------------ |
| Estabilidad             | Alta               |
| Dirección tonal         | Punto de llegada   |
| Necesidad de resolución | Ninguna            |
| Grados típicos          | Imaj7, VI-7, III-7 |
| Ejemplos en C mayor     | Cmaj7, Am7, Em7    |

### Función subdominante

| Rasgo                   | Detalle                  |
| ----------------------- | ------------------------ |
| Estabilidad             | Intermedia               |
| Dirección tonal         | Alejamiento de la tónica |
| Necesidad de resolución | Moderada                 |
| Grados típicos          | II-7, IVmaj7             |
| Ejemplos en C mayor     | Dm7, Fmaj7               |

### Función dominante

| Rasgo                   | Detalle         |
| ----------------------- | --------------- |
| Estabilidad             | Baja            |
| Dirección tonal         | Hacia la tónica |
| Necesidad de resolución | Alta            |
| Grados típicos          | V7, VIIø7       |
| Ejemplos en C mayor     | G7, Bm7♭5       |

## Resumen funcional por grado (C mayor)

| Grado | Acorde | Función         |
| ----- | ------ | --------------- |
| I     | Cmaj7  | Tónica          |
| II    | Dm7    | Subdominante    |
| III   | Em7    | Tónica débil    |
| IV    | Fmaj7  | Subdominante    |
| V     | G7     | Dominante       |
| VI    | Am7    | Tónica          |
| VII   | Bm7♭5  | Dominante débil |

## Conclusión

El uso de cuatriadas permite:

- Identificar con mayor precisión las funciones armónicas.
- Incorporar la séptima como elemento clave en la clasificación funcional.
- Construir progresiones más sofisticadas con movimiento armónico intencionado.

La función armónica aquí presentada corresponde a la escala mayor. Otras escalas, como la menor relativa, tienen un sistema funcional distinto, ya que su enfoque es modal y se aborda en niveles más avanzados.
