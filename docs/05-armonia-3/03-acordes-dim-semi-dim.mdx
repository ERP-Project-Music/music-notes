# Acordes Disminuidos y Semidisminuidos

Los **acordes disminuidos (°7)** y **semidisminuidos (ø7)** son cuatriadas que aportan una gran tensión armónica, lo que los hace muy útiles como acordes de paso o dominantes sustitutos.

A diferencia de otras cuatriadas, estos acordes tienen una **estructura simétrica o inestable**, lo que les da un fuerte impulso hacia la resolución.

## Acorde disminuido (°7)

**Estructura:**

- Tónica
- Tercera menor
- Quinta disminuida
- Séptima disminuida

**Ejemplo:** B°7  
Notas: B - D - F - Ab

Este acorde tiene una **estructura simétrica** (intervalos de terceras menores), lo que lo hace especialmente versátil para sustituir funciones dominantes.

### Función típica:

- Se usa como **dominante sustituto** del V7.
- El **vii°7** en una tonalidad mayor funciona como dominante del I.

**Ejemplo (C mayor):**  
B°7 → Cmaj7

## Acorde semidisminuido (ø7)

**Estructura:**

- Tónica
- Tercera menor
- Quinta disminuida
- Séptima menor

**Ejemplo:** Bø7  
Notas: B - D - F - A

Este acorde aparece naturalmente en el **VII grado de la escala menor natural** (por ejemplo, en C menor: Bø7).

### Función típica:

- Funciona como **subdominante menor** o como **acorde de paso** hacia un dominante.
- Muy común en la progresión **iiø7 - V7 - Im7** en el modo menor.

### Ejemplo (A menor)

Bm7♭5 → E7 → Am7

## Diferencias clave

| Característica       | Disminuido (°7)          | Semidisminuido (ø7)      |
| -------------------- | ------------------------ | ------------------------ |
| Séptima              | Séptima disminuida       | Séptima menor            |
| Función común        | Dominante sustituto      | Subdominante o paso      |
| Presencia en escalas | Menor armónica/melódica  | Menor natural y modos    |
| Resolución típica    | Directamente a la tónica | Hacia dominante o tónica |

## Aplicaciones armónicas

- Puedes usar el °7 en lugar de un V7 para una resolución más tensa.
- El ø7 permite progresiones suaves en modo menor o entre acordes secundarios.
- Por su simetría, el °7 se puede **transponer por terceras menores** sin perder función, ideal para movimientos cromáticos.

### Ejemplo de uso (C mayor):

Em7 - A7 - Dm7 - G7 - B°7 - Cmaj7
