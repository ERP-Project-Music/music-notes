# Acordes con extensiones

En armonía, los **acordes con extensiones** (también llamados _extended chords_) son aquellos que añaden notas más allá de la séptima: la novena (9ª), la oncena (11ª) y la trecena (13ª).

Estas notas se obtienen al seguir apilando terceras sobre una cuatriada.

Es importante recalcar que la extensión para que sea valida la triada o cuatriada debe estar construida correctamente.

## ¿Qué son las extensiones?

Las extensiones se basan en continuar la construcción del acorde más allá de la séptima:

| Extensión | Nota base | Explicación                     |
| --------- | --------- | ------------------------------- |
| 9ª        | 2ª        | Es la 2ª nota una octava arriba |
| 11ª       | 4ª        | Es la 4ª nota una octava arriba |
| 13ª       | 6ª        | Es la 6ª nota una octava arriba |

Estas notas amplían el color y la complejidad del acorde.

## Triadas con extensiones (Acordes Add)

Cuando una **triada** (mayor o menor) añade una nota superior sin incluir la séptima, se llama acorde **Add** seguido del número correspondiente. Esto ocurre comúnmente con la **9ª** (add9), pero puede extenderse a otras.

**Ejemplos:**

| Tipo de triada    | Nombre del acorde | Notas          | Fórmula         |
| ----------------- | ----------------- | -------------- | --------------- |
| Mayor con novena  | Cadd9             | C - E - G - D  | 1 - 3 - 5 - 9   |
| Menor con novena  | Amadd9            | A - C - E - B  | 1 - ♭3 - 5 - 9  |
| Mayor con oncena  | Cadd11            | C - E - G - F  | 1 - 3 - 5 - 11  |
| Menor con oncena  | Amadd11           | A - C - E - D  | 1 - ♭3 - 5 - 11 |
| Mayor con trecena | Cadd13            | C - E - G - A  | 1 - 3 - 5 - 13  |
| Menor con trecena | Amadd13           | A - C - E - F♯ | 1 - ♭3 - 5 - 13 |

## Cuatriadas con extensiones

Cuando a una **cuatriada** (acorde con séptima) se le añaden extensiones como la novena, oncena o trecena, el acorde **cambia de nombre** para reflejar la nueva tensión incorporada.

| Acorde base | Extensión añadida | Nombre final |
| ----------- | ----------------- | ------------ |
| Cmaj7       | 9ª                | Cmaj9        |
| C7          | 9ª                | C9           |
| Cm7         | 9ª                | Cm9          |

**Ejemplos:**

| Acorde | Notas       | Comentario                            |
| ------ | ----------- | ------------------------------------- |
| Cmaj9  | C E G B D   | Cuatriada mayor + 9ª                  |
| Cm9    | C Eb G Bb D | Cuatriada menor + 9ª                  |
| Cmaj11 | C E G B F   | Cuatriada mayor + 11ª (sin 9ª)        |
| Cm11   | C Eb G Bb F | Cuatriada menor + 11ª (sin 9ª)        |
| Cmaj13 | C E G B A   | Cuatriada mayor + 13ª (sin 9ª ni 11ª) |
| Cm13   | C Eb G Bb A | Cuatriada menor + 13ª (sin 9ª ni 11ª) |

### Omite la quinta si es necesario

Cuando se tocan acordes con muchas notas (como los extendidos), **se suele omitir la quinta** por razones prácticas, especialmente en guitarra o arreglos orquestales. La quinta no aporta tanta información armónica como la tercera, séptima o la extensión, así que es común eliminarla para facilitar la digitación o evitar un sonido sobrecargado.
