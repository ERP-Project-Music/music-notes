# Progresiones

Una **progresión armónica** es una secuencia de acordes que crea movimiento y dirección dentro de una pieza musical. Las progresiones definen el carácter y la tensión de una canción.

## Tipos comunes de progresiones

| Función            | Grados          | Comentario                                                           |
| ------------------ | --------------- | -------------------------------------------------------------------- |
| **T → SD → D → T** | I - IV - V - I  | Progresión clásica tonal.                                            |
| **SD → D → T**     | II - V - I      | Progresión jazz por excelencia, típica resolución fuerte.            |
| **T → SD → D → T** | I - II - V - I  | Tónica fuerte seguida de preparación (**II**) y resolución.          |
| **T → SD → D → T** | VI - II - V - I | El **VI** funciona como tónica relativa que abre hacia el **II**.    |
| **T → D → T**      | I - VII° - I    | El **VII°** resuelve directamente a la tónica, como dominante débil. |

## Ejemplos en C mayor

| Progresión        | Acordes        |
| ----------------- | -------------- |
| I - IV - V - I    | C - F - G - C  |
| I - vi - IV - V   | C - Am - F - G |
| vi - IV - I - V   | Am - F - C - G |
| I - V - vi - IV   | C - G - Am - F |
| vi - V - IV - III | Am - G - F - E |

## Conclusión

Comprender y practicar progresiones armónicas te permite:

- Construir acompañamientos efectivos.
- Identificar patrones en canciones.
- Componer con estructura y dirección.
