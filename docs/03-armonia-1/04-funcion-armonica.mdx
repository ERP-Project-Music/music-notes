# Funciones armónicas

En la armonía tonal, cada acorde cumple una función dentro de la tonalidad. Estas funciones se agrupan en tres categorías:

- **Tónica**: zona de reposo o resolución.
- **Subdominante**: zona de transición o preparación.
- **Dominante**: zona de tensión que tiende a resolverse.

## Clasificación funcional en la tonalidad mayor

| Función      | Grados     | Ejemplos en C mayor |
| ------------ | ---------- | ------------------- |
| Tónica       | I, VI, III | C, Am, Em           |
| Subdominante | II, IV     | Dm, F               |
| Dominante    | V, VII°    | G, Bdim             |

## Características generales por función

### Función tónica

| Rasgo                   | Detalle          |
| ----------------------- | ---------------- |
| Estabilidad             | Alta             |
| Dirección tonal         | Punto de llegada |
| Necesidad de resolución | Ninguna          |
| Grados típicos          | I, VI, III       |
| Ejemplos en C mayor     | C, Am, Em        |

### Función subdominante

| Rasgo                   | Detalle                  |
| ----------------------- | ------------------------ |
| Estabilidad             | Intermedia               |
| Dirección tonal         | Alejamiento de la tónica |
| Necesidad de resolución | Moderada                 |
| Grados típicos          | II, IV                   |
| Ejemplos en C mayor     | Dm, F                    |

### Función dominante

| Rasgo                   | Detalle         |
| ----------------------- | --------------- |
| Estabilidad             | Baja            |
| Dirección tonal         | Hacia la tónica |
| Necesidad de resolución | Alta            |
| Grados típicos          | V, VII°         |
| Ejemplos en C mayor     | G, Bdim         |

## Resumen funcional por grado (C mayor)

| Grado | Acorde | Función         |
| ----- | ------ | --------------- |
| I     | C      | Tónica          |
| II    | Dm     | Subdominante    |
| III   | Em     | Tónica débil    |
| IV    | F      | Subdominante    |
| V     | G      | Dominante       |
| VI    | Am     | Tónica          |
| VII°  | Bdim   | Dominante débil |

## Conclusión

Dominar las funciones armónicas permite:

- Comprender cómo se organizan los acordes dentro de una tonalidad.
- Analizar y construir progresiones coherentes.
- Anticipar resoluciones armónicas y diseñar acompañamientos con intención.

Esta función armónica se basa en la escala mayor. Cada escala tiene su propio sistema funcional. La relativa menor no comparte las mismas funciones, ya que responde a un enfoque modal, lo cual se estudia más adelante.
