# Modo Locrio

El **modo locrio** es el **séptimo modo** de la escala mayor. Es el más inestable y disonante de todos, ya que contiene una **segunda menor (♭2)** y una **quinta disminuida (♭5)**. Su sonido es **oscuro, tenso y caótico**, por lo que raramente se usa como centro modal principal, aunque puede funcionar en pasajes breves o secciones experimentales.

## Característica sonora

- Sonido **inquietante, inestable y tenso**.
- No tiene acorde de tónica estable ni dominante funcional.
- Es ideal para atmósferas oscuras o abstractas.

## Fórmulas y estructura del modo locrio

| Tipo                    | Fórmula                        | Ejemplo                   |
| ----------------------- | ------------------------------ | ------------------------- |
| **Escala**              | 1 - ♭2 - ♭3 - 4 - ♭5 - ♭6 - ♭7 | B - C - D - E - F - G - A |
| **Arpegio (cuatriada)** | 1 - ♭3 - ♭5 - ♭7               | B - D - F - A (Bø7)       |

### ¿Qué hace que suene a locrio?

- Tiene una **segunda menor** que genera disonancia inmediata.
- La **quinta disminuida** impide la estabilidad del acorde tónico.
- El acorde base es un **semidisminuido (ø7)**, difícil de usar como centro estable.
- Generalmente se usa como **pasaje o color oscuro**, no como modo principal.

## Progresiones usuales

- Bø7 - Dm7 - G7 - Cmaj7  
  (Movimiento desde la tensión hacia resolución tonal)

- Bø7 - Fmaj7 - Gm7 - Bø7  
  (Secuencia modal oscura, sin resolución clara)

- Bø7 - C - Bø7  
  (Color locrio puro con acercamiento cromático)

Estas progresiones aprovechan la tensión inherente del **modo locrio**, que rara vez se sostiene por mucho tiempo pero ofrece un **color dramático** muy característico.
